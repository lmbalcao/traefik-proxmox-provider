# Security Policy

## Reporting A Vulnerability

If you discover a security issue in this repository or its automation:

- do not open a public issue with exploitable details
- report it privately to the repository owner or maintainers through an agreed private channel
- include impact, affected files, reproduction steps, and suggested remediation if known

## Supported Scope

Security fixes are handled on a best-effort basis for:

- the default branch
- the latest released version when release automation is used
